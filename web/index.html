Introduction

This page runs a model of a ridehail system, such as Uber or Lyft, in a city. The model is written in python. It runs in the browser using the Pyodide distribution, the UI is built with Material Design Lite, and the page takes inspiration from the TensorFlow Playground. They are not to blame for any problems.

Choose your simulation and display options below, and then run the simulation with these controls

Simulation options

In this model, a city is a square grid. Its size is the number of blocks on each side of the square. A block is a measure of both time and distance. You can think of a block as typically about one minute or a "city block" in some North American city.

City size: 24 blocks
